# GasShare - Community LPG Sharing Platform (Uizard Prompt)

Design a mobile-first emergency LPG cylinder sharing platform for middle-class and rural families in India. Clean, modern interface with warm gradients (orange #FF6B35, yellow #FFC857, green #4CAF50) and rounded corners.

**Authentication Flow:**
Multi-step registration with phone OTP verification, LPG passbook validation, and personal details. Login page with OTP system. Include loading states, error messages, and success confirmations.

**Landing Page:**
Hero section: "Never Run Out of Gas Again" with gradient CTA buttons. Three-step visual workflow showing signup, list/request, and exchange process. Trust indicators displaying verified users and ratings. Features section highlighting verified users, 3km emergency radius, community trust scores, and safe exchanges.

**Dashboard:**
User overview displaying circular trust score gauge (0-100) with color coding (green 80+, yellow 60-79, orange below 60). Quick action cards for listing cylinders and finding cylinders with gradient backgrounds. Recent activity feed showing sent/received/pending exchanges with user names, distances, timestamps, and status badges. Notification bell with unread count.

**Sender Page:**
Form with auto-detect location button, availability time picker. Display active listings cards showing location, serial numbers, time slots, and status badges. Map visualization showing nearby requests within 3km radius with pin markers. Safety guidelines panel with verification reminders.

**Receiver Page:**
Urgency selector (Low/Medium/High) with color-coded radio buttons. Search bar with 3km radius indicator. Split view: interactive map showing available senders with distance pins, and scrollable list of sender cards displaying trust scores (numeric with trending icon), star ratings, distance, availability times, and request buttons. Request confirmation modal with sender details and next steps.

**History Page:**
Tabbed interface (All/Received/Sent) with filter dropdown. Transaction cards showing user, location, date, cylinder serial numbers, and completion status. Line chart visualizing trust score growth over six months. Rating dialog with five-star system and review textarea. Statistics showing total exchanges, completed count, and average rating.

Include skeleton loaders, empty states with illustrations, glassmorphism effects, and smooth micro-interactions throughout.
